# Run project
## Start server

Follow the instructions [mws-restaurant-stage-3](https://github.com/udacity/mws-restaurant-stage-3).

## Start application

1. Clone the repo.
2. Install project dependencies.
```
npm install
```
3. Install gulp.
```
npm install gulp-cli -g
```
4. Build project
```
npm run build
```
5. Run project
```
npm start
```
